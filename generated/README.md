# Web3 Wallet Dashboard
## Overview
This is a simple web3 wallet dashboard built with React and Web3.js. It allows users to connect their Ethereum wallet, view their balance, and transfer funds.

## Installation
To run this app, you need to install the following dependencies:
* `web3`: https://unpkg.com/web3@1.2.0/dist/web3.min.js
* `ethers`: https://unpkg.com/ethers@5.4.0/dist/ethers.js

## Running the App
To run this app, navigate to the project directory and run `npm start` or `yarn start`. This will start the development server.
